Це код для 3-ої лаби з криптографії. Він працює з російським текстом, оскільки так просили в методичці.
Текст має бути російською мовою без знаків пунктуації, великих літер та пробілу.
Буква «ё» заміщена буквою «е», а «ъ» – буквою «ь».
Загальна кількість літер у алфавіті m = 31.

Цей код робить:
1. Розшифровує текст зашифрованний афінним шифром. Автоматично перевіряє змістовність тексту, відкидає більшість
неправильних варіантів. Від користувача потрібно лише перевірити відфільтровані варіанти тексту.

This is the code for the 3rd cryptography lab. It works with the Russian text, as it was requested in the manual.
The text must be in Russian without punctuation marks, capital letters and spaces.
The letter "ё" is replaced by the letter "е", and "ъ" - by the letter "ь".
The total number of letters in the alphabet is m = 31.

This code does:
1. Decrypts text encrypted with an affine cipher. Automatically checks the content of the text, rejects most
 incorrect options. The user only needs to check the filtered text variants.
